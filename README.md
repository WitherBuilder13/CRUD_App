# _Main Page_
### 3 Buttons 
- _Add Student_ - add a new student
- _Find Students_ - find all students that match criteria
- _Add Major_ - add a new major
### List of all students
- Shows all students in the database
- Click a student to view all of their info (_Student Info_)
- Long click to delete the student
# _Add Student_
### 7 Fields
- First Name
- Last Name
- Username (must be unique)
- Email
- Age
- GPA
- Major (dropdown)
### 3 Buttons
- _Add Major_
- Back - cancel and return to _Main Page_
- Add - add student to database and return to _Main Page_, the student list will update automatically
# _Student Info_
- Shows all student info
### 2 Buttons
- Back - return to _Main Page_
- Update - modify the current student's info (_Update Student_)
# _Update Student_
- Same as _Add Student_, but the Username cannot be changed
### 3 Buttons
- _Add Major_
- Back - cancel and return to _Student Info_
- Update - save new data and return to _Student Info_, the info will be updated automatically
# Find Students
### 6 Fields
- First Name
- Last Name
- Username
- Major (dropdown)
- Min GPA
- Max GPA
### 1 Button
- Back - return to _Main Page_
### List of all students that meet the criteria
- Updates as criteria are changed
- Click on a student to view their info
- Long click to delete
# _Add Major_
### 2 Fields
- Major Name (must be unique)
- Major Prefix (must be unique)
### 1 Button
- Back - cancel and return to whichever page you came from, the dropdown on that page will update accordingly
